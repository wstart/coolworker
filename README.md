## 快速开始 🚀

1. 克隆仓库：

```bash
# SSH
git clone git@github.com:wstart/coolworker.git
# HTTPS
git clone https://github.com/wstart/coolworker.git
```

2. 安装依赖（根据项目语言/包管理器执行，例如）：

```bash
npm install        # JavaScript/Node 项目
pip install -r requirements.txt  # Python 项目
```

3. 运行测试与本地开发：

```bash
npm test           # 或 pytest、go test 等，视项目而定
npm run start      # 或相应的启动命令
```
